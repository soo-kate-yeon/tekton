## 작성 가이드

### 레퍼런스 선정 기준
- "Linear Minimal" 테마의 톤앤매너와 일치하는 서비스
- 여백, 색상 사용, 타이포그래피가 절제된 디자인
- 접근성(A11y) 고려가 잘 된 UI

### 디자인 원칙 작성 팁
- 구체적으로: "여백 넉넉하게" ❌ → "섹션 간 gap: spacing-8 이상" ✅
- 감성적 표현 포함: "차분하고 신뢰감 있는 분위기"
- 레이아웃 명시: "중앙 정렬", "좌측 사이드바", "전체 너비"

### 금지 사항 예시
- 과도한 그라데이션 배경
- 2개 이상의 CTA 버튼
- 장식적 일러스트레이션
- 하드코딩된 색상값 (#hex, rgb)
